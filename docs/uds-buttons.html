API

属性

属性 说明 类型 默认值
type 按钮类型，决定按钮的颜色和样式 primary | secondary | danger primary
outline 是否使用轮廓样式，设置为true时会应用outline样式 boolean false
text 是否为文本按钮 boolean false
ghost 是否为幽灵按钮，透明背景 boolean false
size 按钮尺寸 small | medium | large md
round 是否使用圆角，默认为直角 boolean false
disabled 是否禁用 boolean false
loading 是否加载中 boolean false
focused 是否处于聚焦状态 boolean false
scale 是否启用按钮点击时的缩放效果 boolean false
counter 设置按钮右侧的计数器数值 string | number -

插槽

名称 说明
默认 按钮内容
icon 按钮左侧图标
icon-right 按钮右侧图标
counter 计数器插槽，用于显示数量信息（推荐使用counter属性代替）
