API 文档

Radio 属性

属性 类型 默认值 说明
checked boolean false 是否选中
disabled boolean false 是否禁用
name string - 表单名称
value string - 单选框的值
type string secondary 类型，可选值：primary, secondary, danger
size string medium 尺寸，可选值：small, medium, large
text boolean true 文本样式（默认样式）
outline boolean false 轮廓样式
ghost boolean false 幽灵样式
filled boolean false 填充样式
counter string - 计数器内容，显示在选项文本右侧
description string - 描述文本，显示在选项文本下方

Radio 事件

事件名 说明 参数
change 选中状态变化时触发 event: Event

RadioGroup 属性

属性 类型 默认值 说明
name string - 表单名称
value string - 当前选中的值
disabled boolean false 是否禁用所有单选框
horizontal boolean false 是否水平排列
type string secondary 类型，可选值：primary, secondary, danger
size string medium 尺寸，可选值：small, medium, large

RadioGroup 事件

事件名 说明 参数
change 选中值变化时触发 value: string
