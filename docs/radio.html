API

Radio 属性

属性 说明 类型 默认值
type 单选框类型，决定单选框的颜色和样式 primary | secondary | danger primary
size 单选框尺寸 sm | md | lg md
text 文本样式（默认） boolean true
ghost 幽灵样式 boolean false
outline 描边样式 boolean false
solid 实心样式 boolean false
disabled 是否禁用 boolean false
checked 是否选中 boolean false
name 单选框名称，通常由RadioGroup统一设置 string -
value 单选框值 string -

RadioGroup 属性

属性 说明 类型 默认值
type 单选框组类型，会应用到所有子单选框 primary | secondary | danger primary
size 单选框组尺寸，会应用到所有子单选框 sm | md | lg md
disabled 是否禁用整个单选框组 boolean false
value 当前选中的值 string -
name 单选框组名称，会应用到所有子单选框 string -
horizontal 是否水平排列 boolean false

事件

事件名称 说明 回调参数
change 选中值变化时触发 event.detail.value: 当前选中的值
