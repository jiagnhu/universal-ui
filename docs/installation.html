安装指南

Universal Design System (UDS) 是一个基于 Web Components 的组件库，可以在任何现代浏览器中使用，无需依赖特定的前端框架。

1安装组件库

使用 npm 安装组件库：

npm install universal-design-system

2在项目中引入组件

方式一：引入整个组件库

在项目入口文件中引入整个组件库：

// 在项目入口文件中引入
import 'universal-design-system';

方式二：按需引入单个组件

只引入需要的组件，减小打包体积：

// 只引入按钮组件
import 'universal-design-system/dist/components/button';

3在HTML中使用组件

引入组件库后，可以直接在HTML中使用组件：

<!-- 基础按钮 -->
<uds-button>默认按钮</uds-button>

<!-- 不同类型的按钮 -->
<uds-button type="primary">主要按钮</uds-button>
<uds-button type="secondary">次要按钮</uds-button>
<uds-button type="danger">危险按钮</uds-button>

<!-- 轮廓按钮 -->
<uds-button type="primary" plain>主要轮廓按钮</uds-button>

<!-- 超小尺寸按钮（适用于移动端） -->
<uds-button size="xs">超小按钮</uds-button>

<!-- 带图标的按钮 -->
<uds-button>
  <svg slot="icon" viewBox="0 0 24 24" width="1em" height="1em">
    <!-- 图标路径 -->
  </svg>
  带图标的按钮
</uds-button>

浏览器兼容性

Universal Design System 支持所有现代浏览器，包括：

  • Chrome (最新版)
  • Firefox (最新版)
  • Safari (最新版)
  • Edge (最新版)

由于使用了 Web Components 技术，不支持 IE11 及以下版本。
