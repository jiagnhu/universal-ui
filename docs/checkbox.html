Checkbox 复选框

Checkbox 复选框用于在一组备选项中进行多选。

基础用法

API

Checkbox 属性

属性 说明 类型 默认值
type 复选框类型，决定复选框的颜色和样式 primary | secondary | danger primary
size 复选框尺寸 sm | md | lg md
text 文本样式（默认） boolean true
ghost 幽灵样式 boolean false
outline 描边样式 boolean false
solid 实心样式 boolean false
disabled 是否禁用 boolean false
checked 是否选中 boolean false
indeterminate 半选状态 boolean false
name 复选框名称，通常由CheckboxGroup统一设置 string -
value 复选框值 string -
description 描述文本，用于提供额外信息 string -

Slots

名称 说明
default 默认插槽，复选框的文本内容
counter 计数器插槽，用于显示数量信息
icon-right 右侧图标插槽

CheckboxGroup 属性

属性 说明 类型 默认值
name 复选框组名称 string -
type 复选框组类型，会应用到所有子复选框 primary | secondary | danger primary
size 复选框组尺寸，会应用到所有子复选框 sm | md | lg md
disabled 是否禁用整个复选框组 boolean false
horizontal 是否水平排列 boolean false
all-checked 是否启用全选功能 boolean false
all-text 全选复选框的文本 string 全选

事件

事件名称 说明 回调参数
change 选中值变化时触发 event.detail.checked: 当前选中状态
event.detail.value: 当前值
